# Password Locker
A terminal run application that allows user to store account credentials

## Author
ESTHER WACHUKA

## Tech Used
 * python3.6
 

### SETUP/INSTALLATION

#### REQUIREMENTS

* Python3.6
* pyperclip
* pip

#### 1. Clone
  * In your terminal run:
    $git clone https://github.com/washucode/Password_Locker
    $cd Password_Locker
#### 2. Run Password_Locker
   * In your terminal:
     $chmod +x runapp.py
     $./runapp.py
#### 3. Test
   * In your terminal:
     $python3.6 usercredentialstest.py
